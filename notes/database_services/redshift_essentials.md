# Databases

## Redshift Essentials

+ Amazon Redshift is a petabyte-scale data warehousing service (OLAP)
+ It is fully managed and scalable 
+ Features 
  + Snapshots
  + Encryption 
  + SQL ( PostgreSQL compatible )
  + Lost Cost (~ $1,000 per TB per year)
+ Massive Parallel Processing (MPP)

+ Generally used for big-data analytics, and it can integrate with most popular business intelligence tools, including Tableau, Jaspersoft, Amazon QuickSight etc.
+ Redshift Spectrum
  + Query unstructured data stored in S3
  + Query exabytes of data in just a few minutes!
